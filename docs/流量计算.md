# 阿里云 ECS 流量使用分析

## 项目流量消耗估算

### 1. 爬虫数据抓取流量

每次抓取（342个城市）：
- PM25.im 页面大小：约 50KB
- 请求头 + 响应头：约 5KB
- 单次抓取总流量：约 55KB

每小时抓取 1 次：
- 每小时：55KB
- 每天：55KB × 24 = 1.32MB
- 每月：1.32MB × 30 = 39.6MB

### 2. 后端 API 流量

假设每天有 100 次访问：
- 单次 API 请求：约 10KB（JSON数据）
- 每天：10KB × 100 = 1MB
- 每月：1MB × 30 = 30MB

### 3. 前端页面流量

用户访问前端页面：
- 页面资源（HTML/CSS/JS）：约 500KB
- 每天 100 次访问：500KB × 100 = 50MB/天
- 但前端通常部署在本地或 CDN，不走服务器流量

### 4. 数据库流量

本地 MySQL，不走公网流量：0MB
